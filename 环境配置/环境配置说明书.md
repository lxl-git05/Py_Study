# 环境配置说明书

## 1. 编码格式

一定要是==UTF-8== !!!

## 2. 创建新环境

- 确定`conda`是否正常:

  - 进入`conda`的命令窗口

  - ```c
    conda --version
    ```

- 设置默认环境目录: 这样以后创建环境就不用写路径了,环境都存在一个文件夹里面

  - ```c
    // 配置为默认
    conda config --add envs_dirs D:\conda_envs
    // 查看是否成功
    conda config --show envs_dirs
    // 第一个是D:\conda_envs就成了
    ```

- ==创建新环境==:

  - ```c
    conda create -n 环境名称 python=版本号(比如3.9.18之类的)
    ```

- 激活环境

  - ```c
    conda activate 环境名称
    ```

## 3. 下载库

* `pip install 包名字`
* `conda install 包名字`
* 区别如下

| 能力         | pip         | `conda` |
| ------------ | ----------- | ------- |
| 解决复杂依赖 | 一般        | 强      |
| 安装速度     | 快          | 稍慢    |
| 环境隔离     | 依赖` venv` | 自带    |
| 跨语言支持   | ❌           | ✅       |
| 体积         | 小          | 大      |
